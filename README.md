# Calculadora de Média Escolar em Python

Sistema desenvolvido em Python para calcular a média escolar de alunos e exibir o resultado final.

## Funcionalidades
- Recebe três notas do aluno
- Calcula a média automaticamente
- Exibe o resultado: Aprovado, Recuperação ou Reprovado
- Permite calcular a média de vários alunos em sequência até encerrar o programa

## Conceitos aplicados
- Entrada e saída de dados
- Variáveis e tipos de dados
- Operadores matemáticos
- Estruturas de decisão com if, elif e else
- Estrutura de repetição com while

## Como executar
1. Certifique-se de ter o Python instalado
2. Clone o repositório
3. Execute o arquivo no terminal com o comando `python calculadora_media_escolar.py`
4. Siga as instruções na tela

## Tecnologias utilizadas
- Python 3
